À propos d’AfriMeteo

AfriMeteo est un tableau de bord météo dédié aux grandes villes africaines. L’objectif est de proposer une lecture claire, rapide et moderne des conditions météo.

Pages 6
Données WeatherAPI
Mode Temps réel

Objectifs

Offrir une interface simple pour consulter la météo actuelle, les prévisions 3 jours et comparer plusieurs capitales africaines.

Fonctionnalités

  • Dashboard temps réel
  • Carte interactive avec capitales
  • Prévisions 3 jours + timeline horaire
  • Comparaison multi‑villes

Sources de données

Les données météo proviennent de WeatherAPI. Les descriptions sont automatiquement traduites en français pour une meilleure lecture.

Charte graphique

Style dashboard sombre inspiré du ciel nocturne, avec accent cyan pour les données clés et orange pour les alertes.
